The above table consits of the scraped financial data used to train the model. The model was not directly trained on this however. The data joined with itself so that each row consisted of eight days of information for each stock, except for the first day of which the model could only see the open price. Additionally, the data was normalized to be between 0 and 1. Then, the data was split into a training set, a validation set, and a test set. The first two sets were randomly shuffled.
