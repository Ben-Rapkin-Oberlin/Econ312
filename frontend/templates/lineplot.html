The Initial Models

Inorder to gauge the effectivness of adding in sentiment analysis, we first trained two models on only the collected finacial data. Initally we planed on only using one model, but out of concern for algorithmic bias, we decided to use two. The first model is a gradient boosted tree from the lightgbm library and the second is a neural network from the sklearn library.

(The black dashed line shows the split between the training/validation data and the test data)

While both networks match major trends in the data, they both struggle with with specific day to day movements. We can see this on the below graph comparing the Mean Squared Error of both graphs.

To give a better idea of what these numbers mean, the MSE for both models on the testing data was ~10e-6, or effectivly 0. Ideally, we want the models to preform as well on the testing sets as the training set. So in otherwords, neither model is doing particularly great. However, that's ok for our purposes. Because we want to look at how sentiment analysis effects their predictions, all we need is a baseline to compare to. Afterall, none of these models are actually going into production. So lets take a look!

The NLP portion of this project is not yet complete, but should be soon!
